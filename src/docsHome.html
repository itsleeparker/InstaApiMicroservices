End Point :

https://apiuptricks.herokuapp.com/api/insta
Queries
Query Name Value
ig_id

Instagram user id generated by Facebook

required
url

Image Media URL to be posted

required
caption

Captions to be posted on the post

access_token

Access Token provided by the facebook graph api for the current user

required
date

Date on which the post is to be published

required NOTE : The Date input format is to be same as given in integrations.
Date can have max of 28 days of difference from the day of posting

Warning

Make very sure the date format is provided same as shown in the eg api call on Postman

Integration

Integration must be done very very carefully , the query can be passed in any order , but the format for date and other variables must be very accurate or same as given api Postman Integration
